# BankPhoneQuickCheck

This project is a WeChat mini app build by wepy. 

## How to run

1. You need to have [wepy](https://tencent.github.io/wepy/) install first.

2. Clone this repo

3. cd to `BankPhoneQuickCheck` flolder

4. run `wepy build`
